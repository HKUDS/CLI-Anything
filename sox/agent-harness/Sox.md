# Sox

A structured CLI harness wrapping SoX audio processing with JSON output, session management, and an interactive REPL.

## Capabilities
- Audio format conversion, bitrate/channel/sample-rate changes
- Trim, concat, mix, reverse, silence removal
- Speed, pitch, tempo, volume, normalize, fade effects
- Audio statistics, spectrogram generation
- Test tone synthesis (sine, square, sawtooth, noise)
- List available SoX effects
