# ImageMagick

A structured CLI harness wrapping ImageMagick `convert`/`magick` with JSON output, session management, and an interactive REPL.

## Capabilities
- Format conversion, resize, crop, rotate, flip, blur, sharpen
- Brightness/contrast adjustment, grayscale conversion
- Watermarks, borders, thumbnails, montages
- GIF animation info, image comparison (RMSE)
- Batch processing with structured JSON output
